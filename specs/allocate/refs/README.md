# Allocate visual reference files

For a **1:1 pixel pass** against the Allocate redesign, keep authoritative HTML in this folder (or update paths here when files move):

- **`allocate-final.html`** — DOM order and interaction reference for `/app/transfers`.
- **`design-system-v3.html`** — Allocate section CSS and tokens (scoped rules, typography, states).

**How to refresh:** Replace the files when design hands off a new version; note the date in your commit message.

If binaries or legal constraints prevent committing HTML, perform the pixel pass locally with files beside this README and treat the **pixel-pass** implementation todo as a manual checklist until copies exist in-repo.
